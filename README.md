Repo de ejemplo
Clonar un repositorio desde Visual Studio Code
1️⃣ Abrir Visual Studio Code

Abrir VS Code en su ordenador.

2️⃣ Abrir el menú de Git

Hacer clic en el icono de Source Control (icono de ramita con puntos) en la barra lateral izquierda.

Alternativamente, usar el menú: Ver → Control de código fuente.

3️⃣ Clonar el repositorio

Hacer clic en Clonar repositorio (o “Clone Repository”).

Pegue la URL del repositorio de GitHub que le diste.

Elegir la carpeta local donde quiere guardar el repositorio.

VS Code descargará el repositorio y preguntará si quieres abrirlo.

Elegir “Abrir” para trabajar directamente en el proyecto.

4️⃣ Configuración inicial (si Git no está configurado)

Si es la primera vez que usa Git:

VS Code pedirá nombre y correo para los commits.

Introducir:

Nombre: "TuNombre"

Correo: "tuemail@example.com"

5️⃣ Editar archivos y guardar cambios

Abrir cualquier archivo, como index.html.

Hacer cambios y guardar (Ctrl + S).

6️⃣ Hacer commit de los cambios

Ir al panel de Source Control.

Verá los archivos modificados listados.

Escribir un mensaje en el recuadro de commit.

Hacer clic en ✔ Commit o en el icono correspondiente.

7️⃣ Subir cambios a GitHub

Hacer clic en … → Push o el botón Sincronizar cambios en la parte inferior.

Esto enviará sus cambios al repositorio remoto en GitHub.

8️⃣ Buenas prácticas

Hacer pull antes de empezar a trabajar: … → Pull para sincronizar cambios.

Hacer commits frecuentes con mensajes claros.
